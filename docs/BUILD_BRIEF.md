# Build Brief

## Product Name

Konfydence Challenge

## Core Promise

Can you spot the scam before it costs you?

## Product Principle

The game must be engaging, but credibility comes first.

The user should feel:

- This was simple.
- This was realistic.
- This could actually happen to me.
- I now know what to do.

## Primary Rule

Consumer / Student:

> Pause → Verify → Ask

Workplace:

> Pause → Verify → Report

TravelSafe:

> Pause → Verify → Use Official Channels

## V1 Scope

Build only the essential challenge experience:

- Scenario
- Four answer choices
- Submit
- Immediate feedback
- Score
- Section progress
- Final result
- Certificate
- Basic admin dashboard

No multiplayer, game board, animations, timers, avatars, or leaderboards in V1.
